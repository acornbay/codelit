# codelit
CodeLit is a modern web application for formatting and exporting code snippets with professional styling. It features syntax highlighting, real-time preview, custom themes, and export options including PNG, JPEG, PDF, HTML, and TXT, making it ideal for coursework, documentation, and presentations.
CodeLit — Code Snippet Formatter and Export Tool

CodeLit is a modern, responsive web application that enables students and developers to format, customise, and export code snippets with a high degree of visual clarity and consistency. It is designed to support academic, professional, and presentation use cases where clean code representation is essential.

Overview

CodeLit provides an intuitive interface for transforming raw code into well-structured, syntax-highlighted snippets. With real-time preview and flexible styling options, users can produce presentation-ready outputs in multiple formats without requiring additional design tools.
